# SuperMission

## 編集手順
コマンドプロンプトかパワーシェルでフォルダ(SuperMission)を開いた状態で

### 最初だけ
1.  $ git clone https://github.com/Minto312/SuperMission.git
2.  ↑で作られたファイルを編集する
3. （編集をネット上に反映させたいとき） $ git commit -a -m "編集内容の説明。とりあえずaaaとかでもいいよ"  
                                  $ git push


### 2回目から
1. $ git pull
2. 編集する
3. $ git commit -a -m "編集内容の説明。とりあえずaaaとかでもいいよ"  
   $ git push
